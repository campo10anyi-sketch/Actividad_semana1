El login es el proceso mediante el cual un usuario ingresa a una aplicación con sus credenciales para verificar su identidad. Gracias al inicio de sesión, el sistema puede permitir el acceso únicamente a las funciones autorizadas para cada usuario. Algunas rutas deben estar protegidas porque contienen información privada o permiten realizar acciones importantes, como modificar datos o administrar el sistema. Si cualquier persona pudiera acceder a ellas, se pondría en riesgo la seguridad de la aplicación. Además, no se deben exponer públicamente contraseñas, documentos de identidad, direcciones, teléfonos, tokens, claves de API ni otros datos sensibles de los usuarios o de la comunidad.
